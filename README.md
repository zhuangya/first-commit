just a helper for showing first commit across different vcs

## Usage

```
 $ first-commit -h

  Usage: first-commit [options]

  Options:

    -h, --help                   output usage information
    -V, --version                output the version number
    -l, --lite                   Lite
    -d, --directory [directory]  Directory
```
